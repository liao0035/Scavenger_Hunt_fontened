## 🏗️ System Architecture

This project is built using a decoupled **Client-Server architecture**. You can find the full source code for both repositories below:

* **Frontend (Current):** [Scavenger_Hunt_fontened](https://github.com/liao0035/Scavenger_Hunt_fontened)
* **Backend API:** [Scavenger_Hunt_backend](https://github.com/liao0035/Scavenger_Hunt_backend)

---

## 🛠️ Tech Stack

### Frontend (Client)
- **Framework:** React.js + Vite
- **Deployment:** Vercel
- **Styling:** Custom CSS & Design System

### Backend (Server)
- **Runtime:** Node.js (Express.js)
- **Database:** MongoDB (Mongoose)
- **Authentication:** Passport.js (Google OAuth 2.0) & JWT
- **Cloud Services:** Google Cloud Storage (Image Processing)
- **Infrastructure:** Redis (Caching), Morgan (Logging), Helmet (Security)
